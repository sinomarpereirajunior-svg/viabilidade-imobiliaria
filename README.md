# Viabilidade Imobiliária

Sistema profissional de análise de viabilidade econômico-financeira para incorporações imobiliárias.

## Funcionalidades

- **Dashboard** — KPIs, TIR, VPL, ROI, margem líquida, exposição de caixa
- **Configurações** — Prazos, CUB, preços, financiamento, velocidade de vendas
- **Urbanismo** — Gabarito, área construída, índices urbanísticos
- **Custos de Obra** — CUB por padrão construtivo, composição detalhada por partidas
- **Despesas** — Incorporação, comercial, administrativo, seguros, financeiro
- **Receitas** — VGV, unidades, permutas, garagens, depósitos
- **CAPM / WACC** — Taxa mínima de atratividade via modelo CAPM ou WACC personalizado
- **Fluxo de Caixa** — Cronograma mensal completo, curvas de desembolso
- **DRE** — Demonstrativo de resultado do exercício
- **Análise de Risco** — Cenários otimista/base/pessimista, sensibilidade
- **Indicadores** — Benchmark setorial, critérios de aprovação configuráveis
- **Relatórios** — Exportação para impressão PDF

## Tecnologia

- HTML5 / CSS3 / JavaScript puro (single-file app)
- [Chart.js 4.4.0](https://www.chartjs.org/) via CDN
- Zero dependências de build — abre diretamente no browser

## Uso

Abra `index.html` diretamente no browser ou acesse via URL do deploy.

## Deploy

Hospedado no [Vercel](https://vercel.com) — deploy automático a cada push na branch `main`.
